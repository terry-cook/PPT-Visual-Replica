# Example Artifact License

Synthetic example prompts, conversation summaries, validation reports, and sample metadata are released under the MIT License together with the repository documentation.

Example reference images, generated PNG assets, preview images, and sample PPTX files may be redistributed only when the maintainer has rights to the underlying reference image and generated derivatives. For public examples in this repository, prefer fully synthetic references generated for demonstration.

Before publishing a new example, add a `reference/LICENSE.md` file that states:

- who owns or generated the reference image;
- whether redistribution is allowed;
- whether derived generated assets and PPTX files may be redistributed;
- whether attribution is required.
