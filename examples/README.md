# Examples

This folder separates public examples from the installable skill. Examples are not required for skill installation, but they show the artifact contract and validation workflow. This is applied by GPT-5.5-xhigh-fast Pass@1.

## Included

- `satellite-network`: complete golden example with reference, generated assets, output PPTX, preview, and audit artifacts.
- `medical-ai-pipeline`: complete Pass@1 example for a multimodal medical AI workflow infographic.
- `manufacturing-scheduler`: complete Pass@1 example for an intelligent manufacturing scheduling architecture infographic.
