# Invocation Prompt

```text
使用PPT REPLICA SKILL重新绘制本地图，使用IMAGE GEN提取透明素材，不要自己画矢量图，保持最小语义可编辑
```

Notes:

- IMAGE GEN/imagegen is the default generator for semantic transparent assets.
- Other image APIs may be used only if they accept image inputs and receive the reference image plus relevant crops or residual crops.
- Manually downloaded transparent PNG assets may be stored before the skill run and used as `provided_asset` entries, as long as source/license metadata is recorded.
